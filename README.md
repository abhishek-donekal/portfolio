# Portfolio — Next.js

Personal portfolio built with **Next.js (App Router)**, **Tailwind CSS**, **Framer Motion**, **Three.js** (hero particle field), **Recharts** (skills radar), and **Lucide** icons.

## Prerequisites

- Node.js 20+
- npm (or pnpm / yarn)

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm start
```

## Resume

The CV is served from `/resume.pdf` (`public/resume.pdf`).

## Legacy site

The previous single-file site and notes live under `legacy/` for reference.
